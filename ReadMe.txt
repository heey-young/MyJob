1. Git-2.27.0-64-bit.exe 실행(설치시 사용 에디터 연결)

#######################################################

도스창으로 연결

D:\MyJob(로컬에서 작업할 폴더) 폴더 오른쪽 마우스 클릭시 Git Bash Here 메뉴 선택하면 도스창 열림

D:\MyJob>git init 엔터
Initialized empty Git repository in D:/MyJob/.git/

D:\MyJob>git remote add origin https://github.com/https://github.com/heey-young/template 엔터(로컬저장소와 원격저장소가 연결됨)
